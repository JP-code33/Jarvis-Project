# Jarvis-Project
Hello, this is my first version of Jarvis in which it is very basic and I say that because it wasn't like Jarvis because it didn't use like live data and it was more like a specific type of command was required for it to work. It also didn't have a voice assistance so I didn't feel much like Jarvis.
But building this app was also fun and all like this was my first experience in Java to build something like this. 
This also doesn't have a GUI, so it still runs in any code editor that you have. 
Firstly it will ask you for the password and it is found in the code itself then it will start the system.
You can either say "Hello", ask for "Status" or ask for "Note", "Remind", and "Clean Memory". These are the only feature I have added to this particular version.
I will be adding more to v2 and will hope to also add a GUI to give it a nicer look.
